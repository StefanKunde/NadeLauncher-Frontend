# Map Screenshots

Place CS2 map screenshots here with the following naming convention:

- `de_dust2.jpg`
- `de_mirage.jpg`
- `de_inferno.jpg`
- `de_nuke.jpg`
- `de_overpass.jpg`
- `de_ancient.jpg`
- `de_anubis.jpg`

Recommended dimensions: 800x450 (16:9 aspect ratio)

If a screenshot is missing, the map radar will be shown as a fallback.
